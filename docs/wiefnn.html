Wie funktioniert ein neuronales Netz?

Neuronale Netze, oder auch KNN (künstliches neuronales Netz), werden beim maschinellen Lernen und bei der KI verwendet. KNN sind Algorithmen, die die Funktionsweise des menschlichen Hirns nachahmen. Sie bilden die Grundlage für künstliche Intelligenzen. Neuronale Netze sind in drei Schichten aufgeteilt. Es gibt die Eingabeschicht (Input Layer), die verborgene Schicht (Hidden Layer) und die Ausgabeschicht (Output Layer). Mithilfe von neuronalen Netzen kann eine KI Bilder, Daten, Texte, Geräusche usw. interpretieren. (Unter Ziffern findet man ein Beispiel für eine KI, die Ziffern erkennen kann.) Da das KNN eine Belohnug erhält, wenn es etwas richtig getan hat, also z.B. eine Ziffer richtig bestummen hat, kann sie eine Verbindung in der Hidden Layer stärken, was dann schlussendlich die KI besser macht.

Neuronale Netze sind aus Neuronen aufgebaut. Diese Neuronen sind sozusagen Knoten, die Informationen aufnehmen, modifizieren und wieder ausgeben. Zwischen den einzelen Neuronen sind Verbindungen, und je stärker eine solche Verbindung ist, desto grösser ist die Gewichtung auf den Output. Es gibt immer je nur eine Input und Output Layer, aber dafür gibt es mehrere Hidden Layers.

Input Layer
Die Input Layer gibt wichtige Informationen an das neuronale Netz. Die Neuronan gewichten dann die Daten, bevor sie sie an die Hidden Layer weitergeben. Die Input Layer besteht nur aus einer einzigen Ebene.

Hidden Layer
Die Hidden Layer kann aus mehreren Ebenen bestehen. In dieser Layer werden die Daten, die von der Input Layer kamen wieder neu gewichtet und von einem Neuron zum nächsten Neuron weitergegeben. In jeder Ebene der Hidden layer werden die Daten neu gewichtet. Dies geht so, bis die Output Layer erreicht wird. Während in der Input und der Output Layer die Daten sichtbar sind, kann man sie, wie der Name schon verräht, nicht sehen.

Output Layer
Die Output Layer hat, wie die Input Layer, nur eine Ebene. Diese Ebene schliesst direkt an die letzte Ebene der Hidden Layer an. Die Output-Neuronen geben dann die endgültigen Daten, also Informationen hervor.

Beim Beispiel Ziffern, verwendet die KI ein ähnliches System, wie man in Barcodes benutzt. Schwarze und Weisse stellen. Die KI lernt durch viele, verschiedene, gegebene Beispiele (Daten). Sie analysiert alle Bilder und kann dadurch ihr neuronales Netz stärken. Wenn man eine Ziffer gezeichnet hat und auf "Guess" drückt, werden die schwarzen und weissen Pixel analysiert, und es kommt eine Vermutung heraus, welche aber nicht immer richtig ist. Für die KI ist es sehr schwierig, Handschriften zu erkennen, denn jeder Mensch schribt anderst und somit gibt es auch sehr viele Möglichkeiten, wie eine "3" aussehen könnte.
Wenn man bei Ziffern etwas zeichnen will, was von der KI erkennt werden soll, muss man zuerst einmal auf "Train" drücken, sodass die KI lernt. Danach kann man etwas zeichnen und auf "Guess" drücken. Wenn die KI geraten hat, kann man auf Clear drücken und eine neue Ziffer zeichnen.
