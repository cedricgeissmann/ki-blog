How does a neural network work?

Neural network form the base of deep learning, which is a subfield of machine learning. It is a type of machine learning algorithm that are widely used for a variety of tasks. We'll get to the term "machine learning" in later chapters, but generally speaking, machine learning is the art of learning inspired by the structure of the human brain, which concludes that a neural network mimics the way human brain operates. Neural networks take in data, train themselves to recognize the patterns in the data and then predict the output for a new set of similar data. A Neural networks are made up of layers of neurons, which are the core processing unit of the network. Voice assistants, Forecasting (e.g for weather or stocks) but also music composition can be based on neural networks.
