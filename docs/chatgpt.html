Was ist ChatGPT und wie funktioniert es?

ChatGPT ist ein von OpenAI entwickeltes Programm. Man kann mit ChatGPT ähnlich wie mit einem echten Menschen schreiben und es kann auch von dem Nutzer gestellte Fragen beantworten. Das Programm basiert auf maschinellem Lernen und verwendet das Modell «Generative Pre-trained Transformer» (--> GPT). Dieses Modell wird aufgrund von Sprachmodellen trainiert.

Wenn man ihm eine Frage stellt, analysiert es diese und versucht Daten herauszulesen und einzuordnen. Danach sucht es die passenden Informationen zur Beantwortung und generiert aufgrund von gelernten Sprachmustern eine Antwort.

Was sind die Grenzen von ChatGPT und worauf sollte man bei einer Verwendung achten?

Die Daten, die ChatGPT verwendet sind oftmals inaktuell oder unvollständig. Beispielsweise hat es keine auf den Tag genaue Informationen. Vor allem muss man bei der Benutzung darauf achten, dass ChatGPT ursprünglich hauptsächlich auf englischsprachigen Daten trainiert wurde. Man kann das Programm zwar auch auf deutsch Fragen stellen, das Programm ist einfach weniger darauf trainiert.

Ausserdem spielt es eine Rolle, wie präzise man eine Frage stellt, da ChatGPT einen Text oder eine Frage aufgrund von Trainingsdaten und statistischen Mustern analysiert. Schon nur wenn man eine Frage leicht anders formuliert, kann schon eine andere Antwort dabei rauskommen.

Der Chatbot kann ausserdem eigenständig nicht kontrollieren, ob die ihm zur Verfügung stehenden Daten überhaupt korrekt sind. Man sollte daher bei der Nutzung von ChatGPT immer vorsichtig bleiben und die erhaltenen Daten aufgrund von anderen Quellen überprüfen.

Tipps zur Verwendung mit ChatGPT:

  • Möglichst präzise Fragen Stellen
  • Die Grenzen von ChatGPT im Kopf behalten
  • Die erhaltenen Informationen immer überprüfen
