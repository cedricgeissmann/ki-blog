Was ist KI?

Das Ziel der Künstlichen Intelligenz (KI) ist es, Maschinen und Programme «intelligent» zu machen. Dabei ist das Gehirn das Vorbild für diese Intelligenz. Künstliche Intelligenz soll wie ein Mensch Probleme lösen, Texte verstehen und wiedergeben und Fragen beantworten können.

Wie funktioniert KI?

KI verwendet verschiedene Techniken und Algorithmen (z.B. maschinelles Lernen, neuronale Netze und statistische Modelle), je nachdem welche Aufgabe gestellt wird und wie viele Daten KI aus der Eingabe herausnehmen kann.

Was für einen Einfluss haben Daten auf eine KI?

Daten sind sehr wichtig für eine KI, da diese Programme immer auf Daten basieren. Sie können für das Training und die Entwicklung einer KI positive, aber auch negative Auswirkungen haben.

Mit Daten wird eine KI trainiert, sodass sie immer wie mehr Muster und Regeln erkennen kann. Wenn man daher falsche oder unsorgfältig ausgesuchte Daten bei einer KI für das Training anwendet, besteht die Gefahr, dass die KI ein Bias (ein Vorurteil/eine Verzerrung) entwickelt. Die Korrektheit und Qualität der für eine KI verwendete Daten müssen daher immer auf Qualität und Korrektheit geprüft werden, um allfällige fehlerhafte Reaktionen zu vermeiden. Ausserdem beeinflussen die Daten (zugehörig zum Training), wie schnell eine KI aus neuen Daten lernen kann und folglich wie schnell sich deren Leistung mit der Zeit verbessert.
