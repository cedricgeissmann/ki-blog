Was ist maschinelles Lernen?

Maschinelles Lernen ist, wenn man aus Erfahrung Wissen "künstlich" generiert. Beim maschinellen Lernen analysieren maschinelle Systeme grosse Datensätze, um "künstliches" Wissen zu erlangen. Das machen sie, indem sie zu Algorythmen greifen, um mögliche Regeln, Muster und Gesetzmässigkeiten zu erkennen. Diese werden Sie dann an anderen, unbekannten Daten anwenden, um Vorhersagungen zu treffen.

Maschinelles Lernen besteht aus mehreren Lernarten. Z.B. Unüberwachtes Lernen, Überwachtes Lernen und Verstärkendes Lernen. Beim unüberwachten Lernen erkennt das System versteckte Muster in Daten. Beim überwachten Lernen, macht das System Hervorsagen und Prognosen für unbekannte Daten. Beim verstärkendem Lernen interagieren Algorithmen mit ihrer Umgebung und lernen dann durch ein Belohnungssystem.

Ein weiteres Teilgebiet von maschinelles Lernen ist Deep Learning (tiefes Lernen). Deep Learning spezialisiert sich auf gosse Datenmengen und KNN. Ausserdem kann man Deep Learning gut gebrauchen, wenn man Bilder erkennen will.
