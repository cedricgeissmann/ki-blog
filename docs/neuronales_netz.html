Wie funktioniert ein neuronales Netz?

Ein neuronales Netz ist eine Art von künstlicher Intelligenz, die versucht, das menschliche Gehirn nachzuahmen. Es besteht aus vielen miteinander verbundenen Knoten, die Neuronen genannt werden. Jedes Neuron kann Informationen von anderen Neuronen oder von außen empfangen, verarbeiten und weitergeben. Ein neuronales Netz kann lernen, indem es seine Verbindungen anpasst, um bessere Ergebnisse zu erzielen. Ein Beispiel für ein neuronales Netz ist der Zahlenerkenner von Herr Geissmann.

Zahlenerkenner

Er kann auf einem weißen Feld mit der Hand oder mit dem Stift gezeichnete Zahlen erkennen. Er lernt aus vielen Beispielen, wie die Zahlen aussehen können. Er hat 11 Output-Punkte, die die Zahlen von 0 bis 10 repräsentieren. Er hat viel mehr Input-Punkte, die die verschiedenen Pixel des Feldes darstellen. Die Input-Punkte werden je nach Helligkeit verschieden stark angeregt und geben diese Signale weiter. Die Signale werden in jeder Schicht des Netzes neu gewichtet und verarbeitet, bis sie zum Output kommen. Der Output-Punkt mit der höchsten Aktivität gibt die erkannte Zahl an.

Ein neuronales Netz kann kreativ sein, indem es neue Muster oder Lösungen aus den vorhandenen Daten generiert. Zum Beispiel kann ein neuronales Netz ein neues Bild malen, indem es sich von anderen Bildern inspirieren lässt. Oder es kann einen neuen Text schreiben, indem es die Grammatik und den Stil einer Sprache lernt. Ein neuronales Netz kann auch nicht monoton sein, indem es Abwechslung und Vielfalt in seine Ausgaben einbringt. Zum Beispiel kann ein neuronales Netz verschiedene Witze erzählen oder verschiedene Musikstile komponieren.
