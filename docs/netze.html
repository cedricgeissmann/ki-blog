Neuronale Netze

Sie funktionieren ähnlich wie die funktionsweise des Gehirns. Neuronen verknüpfen im menschlichen Gehirn die Nervenzellen, von welchen es im ganzen Körper Milliarden gibt. Der Computer ahmt diese Neuronalen Netze nach. Informationen werden als Input eingegeben, verarbeitet und das Ergebnis wird auf der anderen Seite als Output ausgegeben. Es entsteht ein Algorithmus. Das bedeutet der Computer findet eine Regel um die Aufgaben richtig zu lösen.
